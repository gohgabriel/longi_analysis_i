What is Frequentist inference?

Building a sampling distribution of parameter values

In the APCES framework, we built up a working theory of how data is generated and how to estimate the quantities we care about. Frequentism is one philosophical tradition that tells us how to interpret those estimates — through tools like sampling distributions, p-values, and confidence intervals. We will walk through these ideas step by step, using the programme evaluation example from the previous chapter: $\theta_{\text{true}}$ represents the true programme effect of time on children's developmental scores.
